Killing Microsoft Defender for Endpoint

What is MsMpLics.dll

MsMpLics.dll file is part of License Module product developed by Microsoft Windows Operating System. This part is required for License Module to function, and without that - both Defender for Endpoint and standard Defender won't work.

Attack Requirements

  • Local Administrator rights

Attack description

This attack allows pentester to disable Microsoft Defender for Endpoint with encrypting dll file.
File can be found in: C:\ProgramData\Microsoft\Windows Defender\Platform\{PLATFORM}\MsMpLics.dll

PoC

  • Navigate to C:\ProgramData\Microsoft\Windows Defender\Platform\
  • Choose the newest platform
  • Find MsMpLics.dll
  • Click Properties → Advanced → Encrypt contents to secure data
  • Save
  • Reboot
